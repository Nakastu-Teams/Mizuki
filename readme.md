# Installation
```
git clone https://github.com/Nakastu-Teams/Mizuki.git
cd Mizuki
npm i 
```
puis configuré le fichier configs.js
# Lancement

### Lavalink 
``` 
yarn lavalink
```
ou
``` 
java -jar lavalink/Lavalink.jar
```

### Mizuki 
``` 
yarn start
```
ou
``` 
node index.js
```
